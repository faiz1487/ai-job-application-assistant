# AI Job Application Assistant

An AI-powered job assistant that parses resumes, matches jobs, generates cover letters, and tracks applications.
Built as a **personal project with startup-level architecture**.

## Features
- Resume parsing using NLP
- AI-based job matching
- AI-generated cover letters
- FastAPI backend
- Dockerized
- CI/CD ready

## Run locally
```bash
docker-compose up --build
```

Open:
http://localhost:8000/docs
